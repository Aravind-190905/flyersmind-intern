## inventory system
an inventory system should keep track of the products that come into a shop and store all the related info.

## features 
one should be able to :
add a product 
remove a product 
update info about product 
search up a specific product 
view all products 
check stock 
exit 

## data structure 
product id integer 
name   str
categoery str
price float 
quantity int

and we also have another class called inventory 

which has all the other features:
add_product()
remove_product()
search_product()
display_all()
save_to_file()
load_from_file()

## flow 

start -> display menu-> perform operation->return to menu again -> exit
